# Regression_to_predict_sales

### This project uses the programming language R and its library dplyr and StepReg to perform linear regression as well as Multi linear regression and step-wise regression to understand weekly walmart sales data to make predication of future sales based on 4 main criterias, which are Consumer Price Index (CPI), Unemployment rate, Average Temperatureof that week and whether it was a calendar holiday.

### The results indicate that Temperature had the most impact on weekly sales however all the other criteria were also significant to predict weekly sales.
